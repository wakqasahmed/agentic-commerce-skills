# Agentic Commerce Skills

Agent skills for ecommerce teams preparing for AI-mediated discovery, recommendation, support, and checkout.

These skills are intentionally short, composable, and practical. They position Waqas Ahmed around Agentic Commerce expertise: SEO, AEO, GEO, AI crawler readiness, product knowledge, policy clarity, and custom-agent remediation.

## Install

```bash
npx skills@latest add wakqasahmed/agentic-commerce-skills
```

Local Claude CLI fallback:

```bash
scripts/list-skills.sh
scripts/link-skills.sh
```

## Skills

- `readiness-audit`: route an ecommerce storefront into Custom Agent, Verified Audit, FDE, or Not Qualified.
- `seo-aeo-geo-audit`: audit discoverability for search engines, answer engines, and generative engines.
- `agent-readiness`: check whether agents can crawl, cite, understand, and act on the storefront.
- `product-knowledge-gap-analysis`: find missing product data that blocks AI answers and support automation.
- `llms-txt-and-crawler-access`: review `robots.txt`, sitemap, `llms.txt`, and crawler rules.
- `ecommerce-policy-readiness`: assess shipping, returns, refunds, warranty, support, and escalation clarity.
- `custom-agent-remediation-plan`: convert audit gaps into a custom-agent implementation plan.
- `fde-opportunity-map`: identify when deeper Forward Deployed Engineering is warranted.

## Positioning

Use this repo for Agentic Commerce and AI-ready ecommerce work. Keep general engineering workflow skills in `wakqasahmed/ai-engineering-workflow-skills`.
